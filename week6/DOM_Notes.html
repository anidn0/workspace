Regina Notes How to Build a DOM

1 set the var of what of the element I want to affect (button etc)

2 check what type of element it is as that changes how I select it.

3 choose my selector (getbyID ...)

4 use my element stored in var to add my desired action with (listener)

5 choose the type of action (click etc) to activate it.

6 set desired external action - (alert, add, remove etc)

7 if dynamically adding, then append to the desired node/element.
